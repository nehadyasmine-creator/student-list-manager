# Student List Manager

Mini projet de gestion de liste d'élèves réalisé en binôme.

## Binômes

- Binôme A : Yasmine (simulé)
- Binôme B : Yasmine (simulation d’un second binôme pour respecter le workflow Git)

## Choix techniques

- Langage : Python
- Organisation :
  - `src/main.py` : CLI
  - `src/student.py` : Student class
  - `src/student_service.py` : gestion des étudiants + export CSV et JSON
- Export : formats CSV et JSON
